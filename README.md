## Variables de entorno

- `DISCORD_BOT_TOKEN`: token del bot de Discord.
- `CLIENT_ID`: ID de la aplicaci√≥n de Discord (para registrar comandos).
- `N8N_WEBHOOK_URL`: webhook de n8n para `/willybot` y `/censura`.
- `WILLYBOT_SECRET` (opcional): secreto compartido para el webhook.
- `ELEVENLABS_API_KEY`: API key de ElevenLabs para `/willytts`.
- `ELEVENLABS_VOICE_ID`: ID de voz de ElevenLabs para `/willytts`.
- `ELEVENLABS_MODEL_ID` (opcional): modelo de ElevenLabs (por defecto `eleven_multilingual_v2`).
- `ELEVENLABS_OUTPUT_FORMAT` (opcional): formato de salida de ElevenLabs (por defecto `opus_48000_64`).
